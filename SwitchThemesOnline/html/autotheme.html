This page is used to generate themes with the Auto-Theme feature.
To create themes use this template :
pageaddr.com/autotheme.html?type=*theme type*&dds=*url to a dds file*
valid theme types are:
home : for home menu background
lock : for lock screen background
user : for user settings page background
the url for the dds parameter must point to a direct download of a 720p DTX1-encoded DDS image
example of a valid url to make a theme for the home menu:
pageaddr.com/autotheme.html?type=home&dds=
